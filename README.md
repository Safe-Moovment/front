
  # Dashboard Prototipo Muu y Saludable

  This is a code bundle for Dashboard Prototipo Muu y Saludable. The original project is available at https://www.figma.com/design/1X955xlPfEJ0CZkdFDAoj4/Dashboard-Prototipo-Muu-y-Saludable.

  ## Running the code

  Run `npm i` to install the dependencies.

  If the backend is not running on `http://localhost:3000`, create a `.env` file in `front/` and set `VITE_API_BASE_URL`.

  Run `npm run dev` to start the development server.
